# 项目规则版本记录

## 当前版本

### v0.8

目标：修复模板可用性问题，形成可直接复制的完整规则包。

主要变化：

- 新增项目特定信息必填区，避免通用规则替代真实项目上下文
- 使用完整目录快照：`AGENTS.md` 与 `docs/30-rules/` 位于同一版本包
- 修复专项规则引用路径
- 增加任务风险分级和高风险操作授权边界
- 明确完成功能所需的测试、配置、迁移和文档属于完整变更集
- 增加不可信内容和提示词注入的信任边界
- 压缩主规则，将低频内容放入专项文件
