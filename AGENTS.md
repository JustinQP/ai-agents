# AGENTS.md

## 1. 仓库定位

本仓库是个人使用的 AI 指令与项目治理模板库，不是运行时 Agent 框架。

## 2. 当前系统入口

- 当前推荐模板：`agent/v1.0/`
- 历史版本：`agent/v0.1` - `agent/v0.9`
- Codex Global：`codex-global/v0.1.md`
- Skills：`agent/v1.0/skills/`

## 3. 规则优先级

在不违反平台安全规则的前提下：

1. 当前任务显式要求
2. 距离目标最近的 AGENTS.md
3. 项目级文档与约定
4. 代码现有实现与测试
5. Codex Global（仅作为行为偏好，不具备覆盖权）

冲突不可自动裁决时必须显式说明，不得隐式选择。

## 4. 修改原则

- 只修改与当前目标直接相关的内容
- 保持最小改动原则
- 不做无关重构或结构性扩展
- 不伪造测试、命令或执行结果
- 涉及风险操作必须说明影响与回滚方案

## 5. Skill 使用原则

- Skill 仅在明确触发时加载
- 不默认读取全部 Skills
- 复杂逻辑由 Skills 承担，AGENTS 仅负责路由与约束边界
