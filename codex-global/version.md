# Codex 全局指令说明

个人全局指令只保存跨项目、跨场景、长期稳定的偏好。

## 大小建议

- 推荐约 500–1200 中文字
- 原则上不超过 2000 中文字

## 适合放入全局指令

1. 希望 AI 扮演的长期角色
2. 默认回答语言和组织方式
3. 复杂问题的拆解方式
4. 通用的风险和真实性要求
5. 常用输出形式

## 不适合放入全局指令

1. 某个项目的技术栈
2. 某个项目的目录结构
3. 具体测试、构建和启动命令
4. 很长的工作流或审计清单
5. 业务背景和临时任务要求
6. 个人账号、密钥或敏感资料

这些内容应放入项目 `AGENTS.md`、专项规则、项目文档或当前对话。
